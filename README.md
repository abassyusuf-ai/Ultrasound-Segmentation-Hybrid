# Ultrasound Segmentation Hybrid Model

## Overview
This project presents a hybrid medical image segmentation approach that combines deep learning techniques with active contour models for improved accuracy in ultrasound image analysis. The model is designed to enhance tumor boundary detection in medical imaging, particularly for cancer diagnosis.

## Problem Statement
Ultrasound images often suffer from:
- Speckle noise
- Low contrast
- Irregular boundaries

These challenges make accurate segmentation difficult using traditional methods. This project addresses these limitations using a hybrid AI-based approach.

## Methodology
- Deep Learning Model (CNN/U-Net architecture)
- Active Contour Model (Chan–Vese or similar)
- Hybrid integration for boundary refinement

## Dataset
- Breast Ultrasound Images Dataset (BUSI) / similar dataset
- Preprocessing includes noise reduction and normalization

## Results
- Improved segmentation accuracy compared to standalone models
- Better boundary detection in low-contrast regions
- Visual outputs demonstrate enhanced tumor localization
